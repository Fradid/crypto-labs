# Crypto Lab — Лабораторія криптографії

Інтерактивна веб-платформа для вивчення та демонстрації роботи класичних та сучасних алгоритмів шифрування. Проєкт поєднує математичну точність алгоритмів з сучасним інтерфейсом та покроковою візуалізацією.

## 🔐 Реалізовані алгоритми

### 📜 Класичні шифри (Заміна та перестановка)
1.  **Шифр Цезаря (Caesar)** — моноалфавітна заміна з циклічним зсувом. Включає модуль **Brute Force** для автоматичного злому.
2.  **Афінний шифр (Affine)** — математична заміна за формулою $C = (aP + b) \mod N$.
3.  **Стовпцевий шифр (Columnar)** — перестановка на основі ключового слова з візуалізацією матриці.
4.  **Rail Fence ("Паркан")** — перестановка символів за зигзагоподібним паттерном.
5.  **Квадрат Полібія (Polybius)** — заміна символів їх координатами у двовимірній матриці.
6.  **Шифр Гронсфельда (Gronsfeld)** — багатоалфавітна заміна з використанням числового ключа.
7.  **Шифр Плейфейра (Playfair)** — біграмний шифр заміни, що використовує матрицю 5x5 або 6x6.

### 🛡️ Сучасні та стійкі методи
8.  **Шифр Вернама (Vernam / OTP)** — ідеально стійкий шифр, що використовує випадковий ключ довжиною в текст (XOR операція).
9.  **DES (Data Encryption Standard)** — реалізація блочного шифру з підтримкою UTF-8 та детальним описом раундів.

### 🔑 Асиметрична криптографія та ЕЦП
10. **RSA Algorithm** — повна реалізація асиметричного шифрування (генерація ключів, шифрування публічним ключем, дешифрування приватним).
11. **Електронний Цифровий Підпис (RSA Sig)** — перевірка цілісності та авторства повідомлень за допомогою хешування **SHA-256** та RSA.

## ✨ Ключові особливості

-   **Жива візуалізація**: Миттєве оновлення результатів при зміні будь-якого символу або ключа.
-   **Покроковий розбір**: Кожен шифр має розділ "Кроки зашифрування" з описом математичних операцій.
-   **UTF-8 Support**: Повна підтримка українського алфавіту, цифр та спеціальних символів.
-   **RSA Key Generator**: Вбудований генератор пар ключів для асиметричних алгоритмів.
-   **Преміальний дизайн**: Темна тема, скляний (glassmorphism) стиль, плавні анімації (Framer Motion).
-   **Responsive UI**: Адаптація під мобільні пристрої та планшети.

## 🛠 Технологічний стек

-   **React 18** + **Vite** — ядро та збірка.
-   **TypeScript** — для надійної роботи з BigInt та типами даних.
-   **Lucide React** — набір сучасних іконок.
-   **Framer Motion** — інтерактивні анімації та мікро-взаємодії.
-   **Vanilla CSS** — кастомна дизайн-система.
